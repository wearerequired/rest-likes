Lightweight solution to like posts and comments, powered by the WordPress REST API.

== Description ==

This highly flexible plugin enables you to like posts, comments, and other object types on your website. Thanks to being powered by the built-in REST API, everything is super fast and easily extensible.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rest-likes` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add a like button to your theme
